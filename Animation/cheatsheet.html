Animation Vocabulary

<frame[, phase[:duration]]> element action [property [from->]to] [easing]
Timeline — When
Frame notation
<2>
Element exists on frame 2 only. No transition — just present during this frame.
 <2->
Element persists from frame 2 onward (Beamer-style range). Once it appears, it stays.
 <2-5>
Element exists on frames 2 through 5 inclusive, then disappears.
 <1,3,5>
Element exists on specific frames only. Beamer overlay spec syntax.
Phases
in
<2, in:500>
<2, in:500> node-A fades ease-in
Entry transition. Plays first when frame starts. Duration = max of all in actions.
 during
<2> or <2, :500>
<2, :500> node-A color white->purple
Main frame phase. Omit phase name for default. Plays after all in actions complete.
 out
<2, out:300>
<3, out:500> node-A disappears
Exit transition. Plays last before advancing to next frame.
Timing
<2, in:800>
Milliseconds. Attached to phase with colon. Phase duration = max of all declared durations.
stagger:N
<2, in:500> nodes q0,q1,q2 fade stagger:100
Same action across a group, offset by N ms between each element.
 ease-in
Starts slow, accelerates. Add at end of any action line.
 ease-out
Starts fast, decelerates. Add at end of any action line.
 ease-in-out
Slow start and end, fast middle. Default easing if unspecified.
 linear
Constant speed throughout. No acceleration.
Playback order
in → during → out → next frame
All in actions play simultaneously, then during, then out. Shorter actions hold until phase completes.
Look — What
Appearance
fades
<2, in:500> node-A fades ease-in
Opacity transition. Fades in during in phase, fades out during out phase.
 appears
<1> node-A appears
Instant show. No transition, element is immediately visible.
 disappears
<3, out> node-A disappears
Instant hide. No transition, element is immediately gone.
 dims
<2> node-A dims
Reduces opacity and/or grays out. Element stays visible but de-emphasized.
Color & Style
color
color [from->]to
<2, :500> node-A color white->purple
Fill or stroke color change. Specify from->to for transition, or just target color.
 fill
<2> node-A fill none->yellow
Fill color change only. Distinct from stroke.
 stroke
<2> edge-B stroke black->red
Stroke/border color change only.
 highlight
<2> node-A highlights
Temporary bright pulse effect, then reverts to original style.
 thickness
thickness [from->]to
<2> edge-A thickness 1->3
Stroke width change. Numeric values in px.
 dashes
<2> edge-A dashes
Change line to dashed or dotted style.
Motion — Where
Spatial
moves
moves (x, y)
<2, :800> node-A moves (3,4) ease-in
Translate element to a new position.
 follows
<2, :1000> node-A follows path-1
Move element along a named path.
 swaps
<2, :500> node-A swaps node-B
Two elements exchange positions simultaneously.
Transform
rotates
rotates [from->]to
<2, :1000> node-A rotates 0->360
Rotation in degrees.
 scales
scales [from->]to
<2, in:500> node-A scales 0->1
Uniform scale factor.
 grows
<2, in:500> node-A grows
Scale from zero + fade in. Combined entry effect.
 shrinks
<2, out:500> node-A shrinks
Scale to zero + fade out. Combined exit effect.
 morphs
<3, out:800> rectangle morphs circle
Shape interpolation between two shapes.
Reveal
draws
<2, in:800> edge-A draws
Stroke-dashoffset animation — looks like a pen drawing the line.
 undraws
<3, out:500> edge-A undraws
Reverse stroke draw — line erases itself.
 types
<2, in:1000> node-A types
Text appears character by character. Typewriter effect.
 wipes
wipes direction
<2, in:500> node-A wipes left-to-right
Directional reveal via clip-path.
Camera
zooms
<3, in:500> camera zooms node-A
ViewBox scales toward a target element. Whole scene effect.
 zooms out
<5, out:500> camera zooms out
ViewBox returns to full view.
 pans
pans (x, y)
<3, :500> camera pans (200, 100)
ViewBox translation. Slides the viewport.
 focuses
<3> camera focuses node-A
Zoom in + dim everything except the target. Spotlight effect.
