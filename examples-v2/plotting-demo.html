Plotting Module Demo

TikZ-compatible function plotting: evaluator + handlers + marks.

Smooth sin(x) with marks

TikZ: \draw plot[smooth, mark=*] function {sin(x)};

Sharp plot: x² parabola

TikZ: \draw plot[domain=-2:2, samples=25] (\x, {\x*\x});

Const plot (staircase)

TikZ: \draw plot[const plot] coordinates {...};

Y-comb (stem plot)

TikZ: \draw plot[ycomb, mark=*] function {sin(x)};

Y-bar chart

TikZ: \draw plot[ybar] coordinates {...};

Parametric: circle

TikZ: \draw plot[smooth, variable=\t, domain=0:360] ({cos(\t)},{sin(\t)});
